# Domain: ci_cd

## Purpose
Contains knowledge about CI/CD workflows, build automation, and release processes for Rocket.Chat Electron desktop application

## Scope
Included in this domain:
- GitHub Actions workflows
- Build and bundling processes
- Platform-specific build configurations (macOS, Windows, Linux)
- Code signing and notarization
- Release automation

Excluded from this domain:
- Application runtime behavior
- UI component implementation
- User-facing features
