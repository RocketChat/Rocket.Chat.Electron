# Topic: app_module

## Overview
Covers application bootstrap, Redux state foundation, and settings management for the Rocket.Chat Electron app.

## Key Concepts
- APP_SETTINGS_LOADED hydration action
- OverrideOnlySettings for admin-only configuration
- Hardware acceleration control
- Platform-specific switches (GPU, Wayland, RDP)
- NTLM credential domain allowlisting
