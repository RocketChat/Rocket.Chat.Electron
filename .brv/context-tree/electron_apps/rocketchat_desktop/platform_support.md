## Raw Concept
**Task:**
Define platform support matrix for Rocket.Chat.Electron

**Changes:**
- Windows 10+ with x64, ia32, arm64 architectures
- macOS 12+ with Universal (x64 + Apple Silicon) architecture
- Linux Ubuntu 22.04+ with x64 architecture

**Timestamp:** 2026-04-04

## Narrative
### Structure
Three primary platforms supported: Windows, macOS, and Linux. Each platform has specific minimum versions, supported architectures, and installer formats.

### Highlights
Windows supports 3 architectures, macOS supports Universal binary for both Intel and Apple Silicon, Linux requires Ubuntu 22.04+

### Rules
Windows minimum version: 10
macOS minimum version: 12 (Monterey)
Linux minimum version: Ubuntu 22.04 or equivalent
