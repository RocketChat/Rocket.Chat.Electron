## Narrative
### Structure
Desktop application wrapping Rocket.Chat web app in Electron shell. Available for macOS, Windows, and Linux. Provides native desktop features and multi-server management.

### Dependencies
Electron framework, Rocket.Chat web client

### Highlights
Notifications, screen sharing, deep links, spell checking, auto-updates, Jitsi integration, Outlook calendar sync, multi-server management
